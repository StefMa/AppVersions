# App Versions
A web app which shows you the current version of apps in the resp. app stores

## Examples
* [Android (single app) - ioki Hamburg](https://appversions.vercel.app/?android=com.ioki.hamburg)
* [Android (multiple apps) - ioki Hamburg, ioki Wittlich](https://appversions.vercel.app/?android=com.ioki.hamburg,com.ioki.wittlich)
* [iOS (single app) - ioki Hamburg](https://appversions.vercel.app/?ios=ioki-hamburg/id1400408720)
* [iOS (multiple apps) - ioki Hamburg, ioki Wittlich](https://appversions.vercel.app/?ios=ioki-hamburg/id1400408720,ioki-wittlich/id1377071496)
* [Android & iOS (multiple apps) - ioki Hamburg, ioki Wittlich](https://appversions.vercel.app/?android=com.ioki.hamburg,com.ioki.wittlich&ios=ioki-hamburg/id1400408720,ioki-wittlich/id1377071496)
